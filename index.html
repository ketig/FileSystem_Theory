C# File Stream рЃЏрЃћрЃюрЃбрЃЮрЃарЃўрЃюрЃњрЃўрЃА рЃАрЃћрЃАрЃўрЃљ

рЃюрЃљрЃгрЃўрЃџрЃў II - рЃцрЃљрЃўрЃџрЃћрЃЉрЃЌрЃљрЃю рЃЊрЃљ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ

FileInfo/File рЃЊрЃљ DirectoryInfo/Directory рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў

­ЪЊЁ рЃАрЃћрЃАрЃўрЃўрЃА рЃњрЃћрЃњрЃЏрЃљ (120 рЃгрЃБрЃЌрЃў) Рќ╝

рЃЊрЃарЃЮрЃўрЃЌрЃў рЃњрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃљ:

  • 0-10 рЃгрЃБрЃЌрЃў: рЃњрЃљрЃЏрЃћрЃЮрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў
  • 10-30 рЃгрЃБрЃЌрЃў: FileInfo рЃЊрЃљ File рЃЎрЃџрЃљрЃАрЃћрЃЉрЃўрЃА рЃЌрЃћрЃЮрЃарЃўрЃљ
  • 30-50 рЃгрЃБрЃЌрЃў: DirectoryInfo рЃЊрЃљ Directory рЃЎрЃџрЃљрЃАрЃћрЃЉрЃўрЃА рЃЌрЃћрЃЮрЃарЃўрЃљ
  • 50-60 рЃгрЃБрЃЌрЃў: рЃерЃћрЃАрЃЋрЃћрЃюрЃћрЃЉрЃљ
  • 60-100 рЃгрЃБрЃЌрЃў: рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў
  • 100-120 рЃгрЃБрЃЌрЃў: рЃАрЃљрЃерЃўрЃюрЃљрЃЮ рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃћрЃЉрЃљ

рЃАрЃћрЃАрЃўрЃўрЃА рЃЏрЃўрЃќрЃюрЃћрЃЉрЃў:

  • FileInfo рЃЊрЃљ File рЃЎрЃџрЃљрЃАрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃўрЃА рЃњрЃљрЃњрЃћрЃЉрЃљ
  • DirectoryInfo рЃЊрЃљ Directory рЃЎрЃџрЃљрЃАрЃћрЃЉрЃўрЃА рЃљрЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљ
  • рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃЏрЃљрЃюрЃўрЃърЃБрЃџрЃљрЃфрЃўрЃўрЃА рЃбрЃћрЃЦрЃюрЃўрЃЎрЃћрЃЉрЃўрЃА рЃЊрЃљрЃБрЃцрЃџрЃћрЃЉрЃљ
  • рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃћрЃЉрЃљ
­ЪЊџ рЃЌрЃћрЃЮрЃарЃўрЃБрЃџрЃў рЃюрЃљрЃгрЃўрЃџрЃў Рќ╝

FileInfo vs File рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў

рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃћрЃЉрЃў:

рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљ FileInfo File
рЃбрЃўрЃърЃў Instance рЃЎрЃџрЃљрЃАрЃў Static рЃЎрЃџрЃљрЃАрЃў
рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃћрЃарЃЌ рЃцрЃљрЃўрЃџрЃЌрЃљрЃю рЃЏрЃарЃљрЃЋрЃљрЃџрЃ»рЃћрЃарЃљрЃЊрЃў рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃћрЃарЃЌрЃ»рЃћрЃарЃљрЃЊрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃАрЃўрЃЕрЃЦрЃљрЃарЃћ рЃБрЃцрЃарЃЮ рЃАрЃгрЃарЃљрЃцрЃў рЃЏрЃарЃљрЃЋрЃљрЃџрЃ»рЃћрЃарЃљрЃЊрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃБрЃцрЃарЃЮ рЃАрЃгрЃарЃљрЃцрЃў рЃћрЃарЃЌрЃ»рЃћрЃарЃљрЃЊрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃЏрЃЮрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃЏрЃћрЃбрЃў рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃљ рЃюрЃљрЃЎрЃџрЃћрЃЉрЃў рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃљ

FileInfo рЃЎрЃџрЃљрЃАрЃў

FileInfo - рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў рЃЊрЃљ рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃћрЃЉрЃў:
  • Name - рЃцрЃљрЃўрЃџрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў
  • FullName - рЃцрЃљрЃўрЃџрЃўрЃА рЃАрЃарЃБрЃџрЃў рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў
  • Length - рЃцрЃљрЃўрЃџрЃўрЃА рЃќрЃЮрЃЏрЃљ рЃЉрЃљрЃўрЃбрЃћрЃЉрЃерЃў
  • CreationTime - рЃерЃћрЃЦрЃЏрЃюрЃўрЃА рЃЊрЃарЃЮ
  • LastWriteTime - рЃЉрЃЮрЃџрЃЮ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮ
  • Exists - рЃцрЃљрЃўрЃџрЃўрЃА рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃљ
  • Directory - рЃЏрЃерЃЮрЃЉрЃћрЃџрЃў рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ
  • Extension - рЃцрЃљрЃўрЃџрЃўрЃА рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃљ
FileInfo - рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў:
  • Create() - рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
  • Delete() - рЃцрЃљрЃўрЃџрЃўрЃА рЃгрЃљрЃерЃџрЃљ
  • CopyTo() - рЃцрЃљрЃўрЃџрЃўрЃА рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃљ
  • MoveTo() - рЃцрЃљрЃўрЃџрЃўрЃА рЃњрЃљрЃЊрЃљрЃбрЃљрЃюрЃљ
  • OpenRead() - рЃЎрЃўрЃЌрЃ«рЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ
  • OpenWrite() - рЃЕрЃљрЃгрЃћрЃарЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ
  • OpenText() - рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃЎрЃўрЃЌрЃ«рЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА
  • CreateText() - рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃЕрЃљрЃгрЃћрЃарЃўрЃАрЃЌрЃЋрЃўрЃА

File рЃЎрЃџрЃљрЃАрЃў

File - рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў:
  • Exists() - рЃцрЃљрЃўрЃџрЃўрЃА рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ
  • ReadAllText() - рЃЏрЃЌрЃћрЃџрЃў рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃЎрЃўрЃЌрЃ«рЃЋрЃљ
  • WriteAllText() - рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃЕрЃљрЃгрЃћрЃарЃљ
  • ReadAllLines() - рЃДрЃЋрЃћрЃџрЃљ рЃ«рЃљрЃќрЃўрЃА рЃЎрЃўрЃЌрЃ«рЃЋрЃљ
  • WriteAllLines() - рЃ«рЃљрЃќрЃћрЃЉрЃўрЃА рЃЕрЃљрЃгрЃћрЃарЃљ
  • Copy() - рЃцрЃљрЃўрЃџрЃўрЃА рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃљ
  • Move() - рЃцрЃљрЃўрЃџрЃўрЃА рЃњрЃљрЃЊрЃљрЃбрЃљрЃюрЃљ
  • Delete() - рЃцрЃљрЃўрЃџрЃўрЃА рЃгрЃљрЃерЃџрЃљ
  • Create() - рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ

DirectoryInfo vs Directory рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў

DirectoryInfo рЃЊрЃљ Directory рЃЎрЃџрЃљрЃАрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃћрЃЉрЃў:

рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљ DirectoryInfo Directory
рЃбрЃўрЃърЃў Instance рЃЎрЃџрЃљрЃАрЃў Static рЃЎрЃџрЃљрЃАрЃў
рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃАрЃЌрЃљрЃю рЃЏрЃарЃљрЃЋрЃљрЃџрЃ»рЃћрЃарЃљрЃЊрЃў рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃћрЃарЃЌрЃ»рЃћрЃарЃљрЃЊрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃАрЃўрЃЕрЃЦрЃљрЃарЃћ рЃБрЃцрЃарЃЮ рЃАрЃгрЃарЃљрЃцрЃў рЃЏрЃарЃљрЃЋрЃљрЃџрЃ»рЃћрЃарЃљрЃЊрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃБрЃцрЃарЃЮ рЃАрЃгрЃарЃљрЃцрЃў рЃћрЃарЃЌрЃ»рЃћрЃарЃљрЃЊрЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА

DirectoryInfo рЃЎрЃџрЃљрЃАрЃў

DirectoryInfo - рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃћрЃЉрЃў:
  • Name - рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў
  • FullName - рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃАрЃарЃБрЃџрЃў рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў
  • Exists - рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃљ
  • Parent - рЃЏрЃерЃЮрЃЉрЃћрЃџрЃў рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ
  • Root - root рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљ
  • CreationTime - рЃерЃћрЃЦрЃЏрЃюрЃўрЃА рЃЊрЃарЃЮ
  • LastWriteTime - рЃЉрЃЮрЃџрЃЮ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮ
DirectoryInfo - рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў:
  • Create() - рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
  • Delete() - рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃгрЃљрЃерЃџрЃљ
  • GetFiles() - рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ
  • GetDirectories() - рЃЦрЃЋрЃћ-рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ
  • MoveTo() - рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃњрЃљрЃЊрЃљрЃбрЃљрЃюрЃљ
  • CreateSubdirectory() - рЃЦрЃЋрЃћ-рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ

Directory рЃЎрЃџрЃљрЃАрЃў

Directory - рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў:
  • Exists() - рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ
